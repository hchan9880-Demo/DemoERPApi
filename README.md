# DemoERPApi

A sample ASP.NET Core Web API project demonstrating:

- REST API development
- SQL Server integration
- Stored Procedures
- Swagger testing
- Customer synchronization workflow

## Technologies

- ASP.NET Core 8
- SQL Server Express
- SSMS
- Swagger/OpenAPI
- C#

## Milestone 1

Features:

- POST /api/customer/sync
- Customer DTO
- Stored Procedure execution
- Customer table
- SyncLog table
